# ⚙️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/Akshay001-A/portfolio.git
```

---

## Navigate To Project

```bash
cd portfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

## Build Production Version

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
